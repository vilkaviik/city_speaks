# Город Говорит

Инструмент для агрегации новостного контента и поиска трендов в новостях при помощи обработки естественного языка

### О проекте
Это мини веб-приложение для отслеживания новостных публикаций из разных источников. Система автоматически собирает контент из заданного списка, анализирует данные с помощью машинного обучения и предоставляет пользователям удобную ленту публикаций. 

### Возможности

- Лента публикаций — хронологический список постов с фильтрами по категории, автору и периоду
- Тренды — популярные темы, обнаруженные путём кластеризации данных

### Технологический стек

**Backend**: FastAPI

**Frontend**: React JS

**База данных**: PostgreSQL

**NLP**: DBSCAN, YandexGPT

### Требования 

- Python 3.10+
- PostgreSQL 14+
- Node.js 18+
- Git
